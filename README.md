
# 🌌 Animated Solar System

Pure HTML, CSS aur vanilla JS se bana ek glowing, animated solar system — Sun se Neptune tak sab planets apni orbit mein realistic speed se ghoomte hain.

## Features
- Sun ka pulsing glow effect
- Har planet apni asli relative speed se orbit karta hai (Mercury fastest, Neptune slowest)
- Earth ka extra tagda blue glow effect
- Earth ke saath ghoomta hua Moon, Saturn ka ring
- Nebula-style colorful sky + twinkling starfield
- **Zoom in/out** (scroll, pinch, ya buttons se) aur **drag to pan**
- **Har planet pe click karo** — uska close-up card khulta hai apne facts ke saath (distance, day length, moons, diameter)
- Pause / Speed-up controls
- Fully responsive + `prefers-reduced-motion` support

## Run locally
Bas `index.html` ko browser mein khol lo — koi build step nahi chahiye.

## Deploy on GitHub Pages
1. Is repo ko GitHub par push karo.
2. **Settings → Pages** mein jao.
3. Source: `main` branch, root folder select karo.
4. Save karo — kuch seconds mein live link mil jayega.

## Tech
- HTML5
- CSS3 (animations, gradients, box-shadow glow)
- Vanilla JavaScript (canvas starfield, zoom/pan, planet modal)

No frameworks, no dependencies, no build tools.
